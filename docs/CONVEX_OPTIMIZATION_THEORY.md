# Convex Optimization Theory

## Mathematical Foundation

### What Makes a Problem Convex?

1. **Convex Objective Function**: A function f is convex if:
   ```
   f(θx + (1-θ)y) ≤ θf(x) + (1-θ)f(y)  for θ ∈ [0,1]
   ```
   Graphically: any line between two points on the graph lies above the graph.

2. **Convex Feasible Set**: The set of points satisfying all constraints forms a convex set.

3. **Convex Constraints**: Inequality constraints g_i(x) ≤ 0 are convex if g_i is convex.

### Portfolio Optimization Problem

**Standard Formulation:**
```
minimize:    w^T * Σ * w          (Portfolio variance - CONVEX!)

subject to:
  μ^T * w ≥ r_target           (Return constraint - LINEAR)
  1^T * w = 1                  (Budget constraint - LINEAR)
  w ≥ 0                        (Non-negativity - CONVEX SET)
```

where:
- w: portfolio weights (decision variables)
- Σ: covariance matrix (positive semi-definite, hence convex)
- μ: expected returns vector
- r_target: target return

### Why This Is Convex

1. **Objective**: w^T * Σ * w is a quadratic form
   - Since Σ is positive semi-definite (covariance matrix), this is convex
   - Second derivative is 2Σ ≽ 0

2. **Constraints**: All linear
   - Linear functions are both convex and concave
   - Linear inequality and equality constraints define convex sets

3. **Feasible Set**: Intersection of:
   - Hyperplane {w: 1^T*w = 1}
   - Half-space {w: μ^T*w ≥ r_target}
   - Orthant {w: w ≥ 0}
   - All convex → intersection is convex

## Key Properties of Convex Optimization

### Local Optimality = Global Optimality

This is THE killer feature. If you find a local minimum of a convex problem, it's guaranteed to be the global minimum!

**Proof sketch**: Assume w* is a local minimum. If it's not global, there exists w' with f(w') < f(w*). By convexity, the line segment between them has f(θw* + (1-θ)w') < f(w*) for small θ > 0, contradicting local optimality.

### Efficient Algorithms Guaranteed to Work

- Interior-point methods: Polynomial time
- Gradient descent: Converges globally
- No risk of getting stuck in local minima

## Algorithms Used

### CVXPY with ECOS Solver

**ECOS** (Embedded Conic Optimizer):
- Interior-point method
- Handles conic constraints
- Handles quadratic objectives (via SOCP transformation)
- Time complexity: O(n³) per iteration, typically 10-30 iterations

**How it works**:
1. Transform problem to conic form
2. Use logarithmic barrier method
3. Solve KKT conditions approximately
4. Iterate until convergence

## Markowitz Mean-Variance Framework

### Efficient Frontier

The efficient frontier is the set of portfolios that:
- Minimize variance for a given target return
- Maximize return for a given variance
- Are not dominated by other portfolios

**Generated by solving**:
```
minimize w^T * Σ * w
subject to:
  μ^T * w = r  (for various r values)
  1^T * w = 1
  w ≥ 0
```

### Capital Market Line (CML)

The line connecting:
- Risk-free asset: (0, r_f)
- Market portfolio: (σ_m, μ_m)

Portfolios on the CML have optimal risk-adjusted returns.

### Maximum Sharpe Ratio Portfolio

**Sharpe Ratio** = (Expected Return - Risk-Free Rate) / Volatility

Maximizing Sharpe ratio is equivalent to finding the portfolio on CML with steepest slope.

**Convex formulation**:
```
minimize:    ||Σ^(1/2) * y||_2
subject to:  (μ - r_f)^T * y = 1
             y ≥ 0

Then w = y / 1^T*y
```

The second-order cone constraint ||Σ^(1/2)*y||_2 ≤ t is convex (SOCP form).

## Computational Example

### Simple 2-Asset Portfolio

Suppose:
- Asset 1: μ₁ = 0.10, σ₁ = 0.15
- Asset 2: μ₂ = 0.15, σ₂ = 0.25
- Correlation: ρ = 0.3
- Target return: r = 0.12
- Risk-free rate: r_f = 0.02

**Covariance matrix**:
```
Σ = [0.15²      0.3*0.15*0.25]
    [0.3*0.15*0.25   0.25²    ]

  = [0.0225    0.01125]
    [0.01125   0.0625 ]
```

**Optimization**:
```
minimize: w₁²*0.0225 + 2*w₁*w₂*0.01125 + w₂²*0.0625

subject to:
  0.10*w₁ + 0.15*w₂ = 0.12
  w₁ + w₂ = 1
  w₁, w₂ ≥ 0
```

**Solution**:
- w₁ ≈ 0.60 (60% in Asset 1)
- w₂ ≈ 0.40 (40% in Asset 2)
- Portfolio σ ≈ 0.195 (19.5%)

## Extensions and Advanced Topics

### Robust Optimization
Handle uncertainty in parameters (μ, Σ)

### Multi-period Optimization
Dynamic portfolio management over multiple periods

### Transaction Costs
Include linear/quadratic costs in objective

### Constraints
- Sector constraints: Σw_i = x% for each sector
- Position limits: 0 ≤ w_i ≤ 0.1
- Turnover constraints: Σ|w_new - w_old| ≤ limit

### Factor Models
Use factor returns instead of individual asset returns
